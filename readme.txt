== Description ==

Repeated Blocks works like a normal post. You can add them to any pages for sections that are repeated such as contact details, newsletter subscription and testimonials.

== Test ==
* Tested with the latest WordPress 4.9.4
* Tested with Visual Composer

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Repeated Blocks"
3. **Activate** Repeated Blocks from your Plugins page
4. **Click** on the new menu item "Blocks" and create your first Block.
5. **Add** them to any pages/posts!
